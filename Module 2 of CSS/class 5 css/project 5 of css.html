Box Model And Float Properties

In today's class we learned about Box model and float properties so lets start with the box model.
Box model
It is a rectangular box which is used for spacing the document.It spaces the document by 4 ways Top,Right,Bottom and Left. It uses three methods:
  1. Margin
  2. Border
  3. Padding
